Link to HTML Forms
This second assignment explored the usage of HTML forms, which can include things like color pickers, text boxes, and checkboxes. This page takes a text input that will make the page say "hello [name]", as well as a color input that will change the background color. There's even a slider that will adjust the text size, and you submit these 3 inputs all at once.
